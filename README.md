# Language Society & Cultural Hub

A premium web platform for language enthusiasts, cultural explorers, and students. Dedicated to fostering global connection through language education and community initiatives.

## Features

- **Multi-lingual Support**: Courses for French, English, German, Urdu, and Arabic.
- **Dynamic Gallery**: Visual showcase of community events and educational impact.
- **Premium UI/UX**: Custom-designed dark theme (Royal Navy Blue) with seamless transitions.
- **Course Registration**: Integrated enrollment system for prospective students.
- **Impact-Driven**: Highlighting the 'Empowering Nation' initiative by founder Iqra Siraj.

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS + Framer Motion
- **Icons**: Lucide React + React Icons
- **Performance**: Optimized WebP asset delivery

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
